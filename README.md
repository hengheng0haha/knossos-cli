# app

> Knossos-cli is Electron-based app. It's build by dva. And it's written by react and antd.

## Dev

Install dependencies.

```bash
$ npm install
```

Start webpack and watch.

```bash
$ npm run dev
```

Then open another terminal, and run start.

```bash
$ npm start
```

## Build

```bash
$ npm run pack
```
